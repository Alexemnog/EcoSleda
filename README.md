# EcoSleda# 🌿 ЕкоСледа v4.0

Въглероден тракер с онлайн база данни Supabase.

## 📁 Структура на проекта

```
ЕкоСледа/
├── main.py          ← Главна точка на влизане
├── app.py           ← Главното приложение (всички табове)
├── auth.py          ← Прозорец за вход и регистрация
├── database.py      ← Всички функции за база данни (Supabase)
├── map_server.py    ← Локален HTTP сървър + Leaflet карта
├── theme.py         ← Цветове, шрифтове и UI хелпъри
├── config.py        ← Supabase URL и ключ
├── ЕкоСледа START.bat     ← 🖥️  Стартер за Windows
```

## 🚀 Как да стартираш

### Windows
Просто кликни двойно на **`ЕкоСледа START.bat`**

### Директно с Python
```bash
python main.py
```

## 📦 Нужни пакети
```
requests
geopy
matplotlib
beautifulsoup4
```

Инсталирай с:
```bash
pip install requests geopy matplotlib beautifulsoup4
```

- 🏠 Начален таб с последна активност
- 🧮 Калкулатор за CO₂ (транспорт + електричество)
- 🗺️ Интерактивна карта с реален маршрут
- 📈 Прогрес с графики
- 🌿 Въздействие върху средата
- 📋 Детайлна история с търсачка
- ⚖️ Сравни два периода
- 🏆 Постижения и значки
- 🎯 CO₂ цели
- 💡 Съвети
- 👥 Общност
- Използва се Python 13.12 версия
- 👤 Профил
